## Funzioni elementari

### Funzioni lineari e affini

- costanti: $$f(x) = c, c \neq 0$$;
- lineari: $$f(x) = mx, m \neq 0$$;
- affini: $$f(x) = mx + q, q \neq 0$$;

### Valore assoluto

$$
|x| = \begin{cases}
x \geq 0\Rightarrow x \\
x < 0 \Rightarrow - x
\end{cases}
$$

**Proprietà**:

- $$|x| \leq c \Leftrightarrow -c \leq x \leq c$$;
- $$|x| < c \Leftrightarrow -c < x < c$$;
- $$|x| \geq c \Leftrightarrow x \leq -c \land x \geq c$$;
- $$|x| > c \Leftrightarrow x < -c \land x > c$$;

Disuguaglianza triangolare: $$|x_{1} + x_{2}| \leq |x_{1}| + |x_{2}|$$,
$$x_{1}, x_{2} \in \R$$

### Potenze e radici

Con $$n \in \N$$:

- $$f(x) = x^{2n}$$:
  - $$D = \R$$, $$Im(f) = [0, +\infty)$$;
  - è una funzione pari;
  - non è nè iniettiva, nè suriettiva;
- $$f(x) = \sqrt[2n]{x}$$:
  - inversa di $$x^{2n}$$;
  - $$D = [0, +\infty)$$, $$Im(f) = [0, +\infty)$$;

---

- $$f(x) = x^{2n + 1}$$:
  - $$D = \R$$, $$Im(f) = \R$$;
  - è una funzione pari;
  - è iniettiva e suriettiva;
- $$f(x) = \sqrt[2n + 1]{x}$$:
  - inversa di $$x^{2n + 1}$$;
  - $$D = \R$$, $$Im(f) = \R$$;

---

$$f(x) = x^{-n} = \frac{1}{x^{n}}$$

---

Quindi, con $$t \in \R$$:

$$
f(x) = x^{t} =
\begin{cases}\begin{align}
t > 1       & \Rightarrow x^{t}           \\
t = 1       & \Rightarrow x               \\
0 < |t| < 1 & \Rightarrow \sqrt[t]{x}     \\
t < 0       & \Rightarrow \frac{1}{x^{-t}}
\end{align}\end{cases}
$$

### Funzioni esponenziali e logaritmiche

$$f(x) = t^{x}$$:

- iniettiva se $$a \neq 1$$, suriettiva se $$Im(f) = (0, +\infty)$$;
- è invertibile da $$\R$$ a $$(0, +\infty)$$;

$$f(x) = \log_{t}{x}$$:

- inversa di $$t^{x}$$;
- strettamente decrescente se $$0 < t < 1$$, strettamente crescente se
  $$t > 1$$;
- $$\log_{t}{1} = 0$$;
- $$\log_{e}{x} = \ln{x}$$;

### Funzioni trigonometriche

#### Seno e coseno

Sul piano cartesiano prendo una circonferenza di centro all'origine e raggio 1.

Sulla circonferenza fisso un punto $$P$$ ad un angolo $$\alpha$$ con l'asse
delle x.

Le coordinate di $$P$$ sono $$(\cos{\alpha}, \sin{\alpha})$$

- $$\sin{\alpha}$$:

  - $$D = \R$$, $$Im(f) = [-1, 1]$$;
  - funzione dispari;

- $$\cos{\alpha}$$:

  - $$D = \R$$, $$Im(f) = [-1, 1]$$;
  - funzione pari;

---

- sono funzioni periodiche: $$T = 2 \pi$$;
- identità fondamentale: $$\sin{(x)}^{2} + \cos{(x)}^{2} = 1$$;

Operazioni:

- addizione:
  - $$\sin{(x \pm y)} = \sin{x}\cos{y} \pm \cos{x}\sin{y}$$;
  - $$\cos{(x \pm y)} = \cos{x}\cos{y} \mp \sin{x}\sin{y}$$;
- duplicazione:
  - $$\sin{2x} = 2(\sin{x}\cos{x})$$;
  - $$\cos{2x} = \cos{(x)}^{2} - \sin{(x)}^{2}$$;

#### Tangente e cotangente

- $$f(x) = \tan{x} = \frac{\sin{x}}{\cos{x}}$$:
  - $$D = \R \setminus \PC{ \frac{\pi}{2} + k \pi \st k \in \Z }$$,
    $$Im(f) = \R$$;
  - periodica: $$T = x + k \pi$$;
- $$f(x) = \cot{x} = \frac{\cos{x}}{\sin{x}}$$;
  - $$D = \R \setminus \PC{ k \pi \st k \in \Z }$$;
  - periodica: $$T = x + k \pi$$;

#### Funzioni inverse

Non sono iniettive perché sono periodiche.

- $$\arcsin{x}$$: invertibile nell'intervallo
  $$[-\frac{\pi}{2}, \frac{\pi}{2}]$$;
- $$\arccos{x}$$: invertibile nell'intervallo $$[0, \pi]$$;
- $$\arctan{x}$$: invertibile nell'intervallo
  $$[-\frac{\pi}{2}, \frac{\pi}{2}]$$;

## Numeri complessi

Risolvono 2 problemi:

- date 2 equazioni di 2° grado:
  - $$x^{2} - x - 2 = 0$$ ha 2 soluzioni reali ($$x_{1} = -1$$ e $$x_{2} = 2$$);
  - $$x^{2} + x + 1 = 0$$ non ha soluzioni reali;
- trovare la radice $$\sqrt[2n]{x}$$ di un numero negativo;

L'insieme dei numeri complessi C è definito come
$$\C = \R \times \R = \PC{ (x, y) \st x \in \R, y \in \R }$$.

Un numero complesso $$z \in \C$$ è una coppia ordinata di numeri reali. In un
numero complesso $$z = (x, y)$$, $$x$$ è la parte reale ($$Re(z)$$) e $$y$$ è la
parte immaginaria ($$Im(z)$$).

- i numeri reali sono coppie ordinate del tipo $$z = (x, 0)$$. Quindi
  i$$\R \subset \C$$;
- i numeri immaginari puri sono coppie ordinate del tipo $$z = (0, y)$$;

Il numero complesso $$(0, 1)$$ è detto unità immaginaria e si denota con la
lettera $$i$$. Il suo valore è pari a $$\sqrt{-1}$$.

2 numeri complessi sono uguali solo se le rispettive parti reali e immaginarie
sono uguali tra loro.

### Operazioni con i numeri complessi

- somma: $$z_{1} + z_{2} = (x_{1} + x_{2}, y_{1} + y_{2})$$;
- moltiplicazione per numero reale: $$t + z = (tx, ty)$$;
- moltiplicazione per numero complesso:

  $$
  \begin{darray}{l}
  z_{1} z_{2} = (x_{1} + iy_{1})(x_{2} + iy_{2})                    = \\
                x_{1} x_{2} + i y_{1} x_{2} + x_{1} i y_{2} + i^{2} y_{1} y_{2} = \\
                x_{1} x_{2} - y_{1} y_{2} + i(y_{1}x_{2} + y_{2}x_1)
  \end{darray}
  $$

### Forma cartesiana di un numero complesso

$$z = (x, y) = x + iy$$.

### Coniugato, modulo e reciproco di un numero complesso

#### Coniugato

Il coniugato $$\overline{z}$$ di un numero complesso $$z$$ è:
$$\overline{z} = x - iy$$.

### Modulo

Il modulo $$|z|$$ di un numero complesso $$z$$ è
$$|z| = |x + iy| = \sqrt{x^{2} + y^{2}}$$.

$$|z|$$ ha le stesse proprietà del modulo di un vettore.

---

Sul piano cartesiano:

- $$\overline{z}$$: riflesso rispetto all'asse x di $$z$$;
- $$|z|$$: modulo del vettore dall'origine a $$z$$;

Quindi $$|z| = |\overline{z}|$$, cambia solo la direzione del vettore quando si
fa il coniugato.

I numeri complessi che hanno modulo 1 sono infiniti e sono tutti i punti che
sono sulla circonferenza di raggio 1 e centro all'origine.

### Proprietà di modulo e coniugato

- $$\overline{z + w} = \overline{z} + \overline{w}$$;
- $$\overline{zw} = \overline{z}\overline{w}$$;
- $$z + \overline{z} = 2 Re(z)$$;
- $$z\overline{z} = |z|^{2}$$;
- $$|zw| = |z||w|$$;
- $$|z| = 0 \Leftrightarrow z = 0$$;
- disuguaglianza triangolare:

  $$|z + w| \leq |z| + |w|$$

  La lunghezza di un lato di un triangolo non è mai maggiore alla somma degli
  altri 2;

:::note

Perché modulo e valore assoluto hanno lo stesso simbolo?

Dato $$z = (x, 0)$$

$$|z| = \sqrt{x^{2} + 0} = \sqrt{x^{2}} = x$$

:::

### Reciproco di un numero complesso

Dato $$z \in \C, \ z \neq 0$$;

$$z^{-1} = \frac{\overline{z}}{|z|^{2}}$$

Il reciproco di un numero complesso è il quoziente tra un numero complesso
($$\overline{z}$$) e un numero reale $$|z|$$.
