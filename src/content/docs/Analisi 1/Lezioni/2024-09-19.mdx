## Funzioni definite a tratti

Il dominio è diviso in parti, e per ogni parte il grafico è descritto da una
funzione diversa.

<details>
<summary>Esempio</summary>

$$
f(x) = |x| = \begin{cases}
x \geq 0 & \Rightarrow x  \\
x < 0    & \Rightarrow -x
\end{cases}
$$

</details>

## Trasformazioni del grafico di una funzione

### Riflessione

- asse $$x$$: $$f(x) \to -f(x)$$;
- asse $$y$$: $$f(x) \to f(-x)$$;

### Traslazione

- verticale: $$f(x) + a$$:
  - se $$a > 0$$: il grafico si sposta verso l'alto;
  - se $$a < 0$$: il grafico si sposta verso il basso;
- orizzontale: $$f(x + a)$$:
  - se $$a > 0$$: il grafico si sposta verso sinistra;
  - se $$a < 0$$: il grafico si sposta verso destra;

### Riscalamento

- verticale: $$k f(x)$$:
  - se $$k > 1$$: il grafico si espande;
  - se $$k < 1$$: il grafico si comprime;
- orizzontale: $$f(k x)$$:
  - se $$k > 1$$: il grafico si comprime;
  - se $$k < 1$$: il grafico si espande;

### Inversa di una funzione

Data $$f: \left[\frac{1}{2}, +\infty\right) \to \R, \ f(x) = x^{2} - x + 1$$;

<Desmos graph="ujadjaawuw" />

Il dominio di $$f^{-1}(x)$$ è dato dal codominio di $$f(x)$$:

$$f^{-1}(x): \left[\frac{3}{4}, +\infty\right) \rightarrow \left[\frac{1}{2}, +\infty\right)$$
